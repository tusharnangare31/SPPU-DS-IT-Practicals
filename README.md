# Distributed Systems Practicals

This repository contains practical implementations of Distributed Systems concepts using Java RMI, CORBA, MPI/OpenMP, Synchronization Algorithms, Leader Election Algorithms, and Web Services.

---

# Practicals List

## 1. RMI Programs

### 1. Cube of a Given Number using RMI
Implement multi-threaded client/server Process communication using RMI to find the cube of a given number.

### 2. Square Root using RMI
Implement multi-threaded client/server Process communication using RMI to find the square root of a given number.

### 3. Square of a Given Number using RMI
Implement multi-threaded client/server Process communication using RMI to find the square of a given number.

### 4. Addition using RMI
Implement multi-threaded client/server Process communication using RMI to find the addition of given numbers.

### 5. Subtraction using RMI
Implement multi-threaded client/server Process communication using RMI to find the subtraction of given numbers.

### 6. Multiplication using RMI
Implement multi-threaded client/server Process communication using RMI to find the multiplication of given numbers.

### 7. Division using RMI
Implement multi-threaded client/server Process communication using RMI to find the division of given numbers.

### 8. Palindrome String using RMI
Implement multi-threaded client/server Process communication using RMI to find whether the string is palindrome or not.

### 9. String Equality using RMI
Implement multi-threaded client/server Process communication using RMI to find whether the string is equal or not.

---

# 2. CORBA Programs

### 10. Calculator Application using CORBA
Develop a Calculator Distributed application using CORBA to demonstrate object brokering.

### 11. String Operation using CORBA
Develop a String Operation Distributed application using CORBA to demonstrate object brokering.

### 12. Odd or Even using CORBA
Develop a distributed application using CORBA to check if the given number is odd or even.

### 13. Prime Number using CORBA
Develop a distributed application using CORBA to check if the given number is prime or not.

---

# 3. MPI / OpenMP Programs

### 14. Sum of N Elements using MPI/OpenMP
Develop a distributed system to find sum of N elements in an array by distributing N/n elements to n number of processors using MPI or OpenMP. Demonstrate by displaying the intermediate sums calculated at different processors.

---

# 4. Synchronization Algorithms

### 15. Berkeley Algorithm
Implement Berkeley algorithm for clock synchronization.

### 16. Token Ring Algorithm
Implement token ring based mutual exclusion algorithm.

---

# 5. Leader Election Algorithms

### 17. Bully Algorithm
Implement Bully algorithm for leader election.

### 18. Ring Algorithm
Implement Ring algorithm for leader election.

---

# 6. Web Services Programs

### 19. Addition Web Service
Create a simple web service and write an addition of two numbers distributed application to consume the web service.

### 20. Subtraction Web Service
Create a simple web service and write a subtraction of two numbers distributed application to consume the web service.

### 21. Multiplication Web Service
Create a simple web service and write a multiplication of two numbers distributed application to consume the web service.

### 22. Division Web Service
Create a simple web service and write a division of two numbers distributed application to consume the web service.

### 23. Sign-Up Form Web Service
Create a simple web service and write a sign-up form distributed application to consume the web service.
